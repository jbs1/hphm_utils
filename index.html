Hello,
This webpage contains utilities for the HP HM mobile game.
I might extend the functionality but this was meant as a little bootstrap exercise so there is no specific plan or anything.
It is a very simply bootstrap webpage hosted on github pages. Since bootstrap is by default responsive it works well enough on Desktop and Phone.
I have not tested it, but given the javascript used in the page it probably only works well on reasonably modern browsers.
The github repo containing the code can be found here: Github Repo

The Puzzling Potions page is a utility to avoid wasting Bottles by opening unnecessary tiles.
When opening one tile you can use this utility to map out the ingredient by referencing the Potion Ingredients Grid from the HP HM Guide from the subreddit. This way you can avoid opening unnecessary tiles and it makes it easier to see where the needed ingredients could possibly be.
This page uses the browsers local storage to save the selected fields between visits. If the last visit is more than 7 days ago it will clear the storage on your next visit for convenience.
Pressing the "Clear all Grids!" button will clear all 4 grids and also clears the local storage.
